In this project, I designed and assembled a custom dynamometer to empirically obtain the motor torque constants of the BLDC motors used in the StaccaToe robot. I fabricated custom motor mounting plates using a CNC mill and assisted in setting up the FUTEK TRS-300 torque sensor in MATLAB for obtaining torque data. The input current to the BLDC motor is measured using the motor's onboard drivers.

Skills used: SolidWorks, CNC, MATLAB
